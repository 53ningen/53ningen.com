## ♨️松島温泉でオタクとエンカ 

つづいては松島温泉、一の坊さんへ。とても大きくて綺麗な旅館。

<a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174243/CB8AAD73-A88F-43C5-8E1E-25A156C595EE.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174243/CB8AAD73-A88F-43C5-8E1E-25A156C595EE-300x225.jpeg" alt="" width="300" height="225" class="alignnone size-medium wp-image-4301" /></a> <a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174148/92FDD3F1-095E-4700-A46A-72A124F9BE2E.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174148/92FDD3F1-095E-4700-A46A-72A124F9BE2E-300x225.jpeg" alt="" width="300" height="225" class="alignnone size-medium wp-image-4291" /></a>

内装もとてもきれい。日帰り立ち寄り入浴でしたが、いつか宿泊したい。

<a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174153/C89F4282-E36A-4008-A326-8869379A3E59.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174153/C89F4282-E36A-4008-A326-8869379A3E59-300x225.jpeg" alt="" width="300" height="225" class="alignnone size-medium wp-image-4292" /></a> <a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174215/384FAA19-9C25-43DA-B9A3-F92A5EB6251E.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174215/384FAA19-9C25-43DA-B9A3-F92A5EB6251E-300x225.jpeg" alt="" width="300" height="225" class="alignnone size-medium wp-image-4296" /></a>

松島温泉、温泉としての期待値をそんなに高く持ってなかったのですが、誇張なしにお湯質がとてもよくて、無色無臭にちかいお湯からは想像できないほどすべすべのお湯でした。今年行った温泉のなかでもベスト3に入るくらいで、温泉地として期待していっても十分期待を超えてくる場所だと思います。

<a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174158/4047A2F0-C644-449F-BD0D-6BD44FBACFDF.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174158/4047A2F0-C644-449F-BD0D-6BD44FBACFDF-300x225.jpeg" alt="" width="300" height="225" class="alignnone size-medium wp-image-4293" /></a> <a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174204/76D79274-9867-4026-9A42-52C1D0E1FCC1.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174204/76D79274-9867-4026-9A42-52C1D0E1FCC1-300x225.jpeg" alt="" width="300" height="225" class="alignnone size-medium wp-image-4294" /></a>

なんと温泉に入って露天風呂でくつろいでいたら、ワグナーさんと遭遇。
**クリスマスイブに裸でエンカする実績を解除しました?**
わぐちゃんにならい風呂上がりにコーヒー牛乳を。

<a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174221/D871C7A8-23F1-412C-9215-540A1B33A282-e1546333497489.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174221/D871C7A8-23F1-412C-9215-540A1B33A282-e1546333497489-225x300.jpeg" alt="" width="225" height="300" class="alignnone size-medium wp-image-4297" /></a>


## ?牡蠣おせんべい「カキとコメと」 

[Wake Up, Miyagi 松島編](https://www.youtube.com/watch?v=LqRSnTvqMaY)にて田中美海さんが訪れた牡蠣おせんべいのお店へ。一の坊から松島海岸駅へ戻る途中にあります。

<a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174254/33C1264F-F247-4A54-80E2-002C55B725EE.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174254/33C1264F-F247-4A54-80E2-002C55B725EE-300x225.jpeg" alt="" width="300" height="225" class="alignnone size-medium wp-image-4303" /></a> <a href="https://static.53ningen.com/wp-content/uploads/2019/01/01174249/7401BDD6-D6F3-44DA-B7BB-9EE5889B5B07.jpeg"><img src="https://static.53ningen.com/wp-content/uploads/2019/01/01174249/7401BDD6-D6F3-44DA-B7BB-9EE5889B5B07-300x225.jpeg" alt="" width="300" height="225" class="alignnone size-medium wp-image-4302" /></a>

その場でおせんべいを焼いてくれて、お酒にあってとてもよいものでした。

観瀾亭も行きたかったのですが、時間の都合で今回は断念...。松島はアクセスもよいため、また次回きたときに訪れようと思い、ゆっくりと松島観光を楽しみました。
